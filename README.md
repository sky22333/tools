## Tools Navigation

一个基于 HTML + Tailwind CSS 的纯静态简约导航站，存放一些我的常用工具。

## 特性
- **零依赖**：无需 Node.js 构建，开箱即用。
- **即时搜索**：支持关键词实时过滤。
- **方便管理**：简单方便的管理域名。
- **智能图标**：自动获取网站头像，多级容错降级。

## 数据管理
修改 `docs/sites.txt` 即可管理内容：
- **添加分类**：`## 分类名称`
- **添加网站**：`网站名称, https://example.com`

## 部署
直接在 GitHub 仓库设置中开启 Pages 服务（Source 选择 `/docs` 目录）即可。
